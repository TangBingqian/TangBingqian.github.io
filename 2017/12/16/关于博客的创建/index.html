关于博客的创建

发表于 2017-12-16 | 分类于 blog | 阅读次数

今天刚刚创建了一个自己的博客，得到倾情帮助，综合参照网上的资料。我这里主要用到的是：Node.js+git+Hexo

网上有很多的教案已经写的很详细了，这里我就不赘述了，就简单的记录一下，以及就自己遇到的一些问题的解决说明一下。

前期配置:

  • 注意Node.js+git+Hexo这三个东西的下载顺序不能乱，Node.js一定要先装，之前装了git也可以，但是一定要是在已经装了Node.js的前提下才可以在git的git bash界面直接下载Hexo，不然总是会出现错误
  • Node.js是直接在官网下载，按照默认配置即可
  • git由于我是之前就装好了的，使用的是git bash界面，这里就不多说了
  • 安装Hexo：

这里搜索的时候出现了两个解决办法：

$npm install hexo-cli -g
$npm install -g hexo

后来查到了:

  • 我安装的版本是最新的，这是hexo之后有的更新，所以这里最新使用的是$npm install hexo-cli -g这个代码，在git bash的窗口打下命令，之后会在对应的本地目录下存在hexo目录

初步尝试:

  • 配置好之后就可以在git bash上进行初步的测试了

有以下基本的hexo语法需要记住:

$hexo n "文件名"   这是在git下创建一个与本地同步的文件名
$hexo g     上传同步
$hexo d     部署
$hexo serve -p 8000  这是将hexo查看的本地IP修改一下，由于系统给出的4000很容易出现被其他软件占用的情况

贴图小技巧

  • 这里我用的是：七牛云+Snipaste+Markdown ，这三个东西配合起来简直好用到爆
  • 注意的是，需要注册一个七牛云的账号，有些麻烦

这里记录一下这三个东西配合使用的一些快捷键:

F1 ：开始截图
ctrl+c :复制图片链接
ctrl+v :直接将图片链接复制到Markdown下就可以展示
shift +双击鼠标左键 ：销毁图片
双击复制到桌面的图片，缩小图片便于整合，再次双击恢复图片大小
  • 注意:不想ctrl + c的时候自动复制图片上传为链接，可以在点击七牛云的图标右键，选择关闭自动上传，就可以向别人发送图片而不是链接了
